## Цель работы: оценить качество работы жесткого диска с помощью специальных программ.
- CrystalDiskMark — это небольшое приложение, которое измеряет скорость записи и чтения устройства. Этот инструмент для проверки дисков отличается от аналогичных утилит тем, что дает возможность использовать разные условия записи и чтения данных, например, измерять показания для блоков разного размера. Также утилита дает возможность установить число проводимых тестов и задать объем данных, используемых для теста.
- С помощью программы CrystalDiskInfo, которая считывает данные SMART, можно определить температуру устройства, количество проблемных секторов, число ошибок позиционирования считывающей головки, время, необходимое на раскрутку диска. Программа также показывает версию прошивки и предоставляет данные о продолжительности использования жесткого диска.

# Контрольные вопросы
## 1. Что такое жесткий диск (HDD)?
- Жесткий диск - Это основной накопитель данных в компьютерах и ноутбуках, основанный на принципе магнитной записи.
- Характеристики Энергонезависимость (данные не исчезают при выключении питания), высокая емкость при относительно низкой стоимости за гигабайт, произвольный доступ к информации. 
## 2. Внутреннее устройство жесткого диска
### Внутреннее устройство включает (вспоминаем ААС):
- Магнитные пластины (Platters): Диски из алюминия или стекла, покрытые ферромагнитным слоем, где хранятся данные.
- Шпиндель (Spindle): Вал, на котором вращаются пластины (обычно 5400-7200 об/мин).
- Головки чтения/записи (Read/Write Heads): Парят над поверхностью пластин на микронном расстоянии.
- Актуатор (Привод головок): Позиционирует головки в нужный сектор.
- Плата электроники (Controller PCB): Управляет механикой и передает данные на интерфейс (SATA/USB).
## 3. Причины возникновения проблем с совместимостью
- Интерфейсы: Использование старых стандартов (IDE/PATA) с новыми материнскими платами (SATA) или отсутствие питания для старых дисков.
- Файловые системы: Форматирование диска в файловой системе, которую не поддерживает операционная система (например, ext4 в Windows, NTFS в старых системах).
- Разметка диска (MBR/GPT): Старые ОС (XP) не видят диски с разметкой GPT, а новые требуют GPT для дисков более 2 ТБ.
- BIOS/UEFI: Несовместимость режимов работы контроллера (AHCI/IDE/RAID).
## 4. Методы уменьшения проблем с совместимостью
- Обновление прошивки (Firmware): Обновление микропрограммы диска или BIOS/UEFI материнской платы.
- Использование адаптеров: Применение переходников (USB-SATA, IDE-SATA) для подключения старых HDD.
- Конвертация разделов: Преобразование диска из MBR в GPT или изменение файловой системы (например, exFAT) для кроссплатформенности.
- Настройка режима контроллера: Установка AHCI в BIOS для современного Windows или IDE для старого оборудования.
